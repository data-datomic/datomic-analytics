Package
DE.SETF.RESOURCE
The de.setf.resource (rdf) package comprises the names for API components of the de.setf.resource linked-data[1][2][3] library. The terms follow from a number of sources: - W3C conceptual models for linked data[4] - Datagraph's RDF.rb and Spira.rb libraries - wilbur - allegrograph - swclos The terms are chosen to reflect the intended operands in that an operator which applies to a certain abstract context or operand class includes a erspective designator as a prefix, while operators which are universal do not. For example the operator rdf:save is universal, while the rdf:save-resource and rdf:save-repository operators intend the respective abstract class of operand. or the statement query operator intended to apply both to resource-object instances and repository instances is named simply 'rdf:has-statement?' The library can be partitioned into several aspects: - declarations - instantiation - query - manipulation property-value, (setf property-value), property-boundp, property-exists-p, property-makunbound, get-statements, insert-statement, or remove-statement . --- [1]: http://www.w3.org/standards/semanticweb/data
