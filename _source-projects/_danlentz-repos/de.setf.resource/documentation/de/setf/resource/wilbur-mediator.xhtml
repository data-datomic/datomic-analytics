This file mediates access to wilbur RDF stores for the `de.setf.resource` CLOS linked data library.
Implement identifier, data, and instance mediation for a wilbur rdf store. - use the subject node instance as the identity cache - wrap the triple acessors - support find for subject based on the node-keyed cache - support select for resource-object and triple instances - delete based on wilbur:triple and rdf:triple instances
